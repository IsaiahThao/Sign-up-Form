# Sign-up Form

A simple, responsive **sign-up form** built with **HTML and CSS** to showcase clean UI design and layout using **Flexbox**. This project is ideal for practicing foundational front-end skills and adding to a beginner web development portfolio.

**Live Demo:** https://isaiahthao.github.io/Sign-up-Form/

---

## Features

- Clean and modern form design  
- Responsive layout using Flexbox  
- Common sign-up fields (name, email, password, etc.)  
- Easy to customize and extend  

---

## Technologies Used

- **HTML** – Structure and form elements  
- **CSS** – Styling and Flexbox layout
